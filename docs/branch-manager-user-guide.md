## Logging In

1. Open the Branch360 Google Sheets application
2. Your account is automatically authenticated using your Google account email
3. The system will recognize your role as Branch Manager
4. You'll be taken directly to your Branch Manager Dashboard

**Note:** If you don't see your dashboard, contact your administrator to ensure your account is set up correctly.

## Viewing Your Dashboard

Your Branch Manager Dashboard provides a comprehensive view of your branch's performance:

### Today's Summary
Real-time snapshot of today's performance:

**Sales Metrics:**
- **TAP**: Total talk-to-appointments across all AEs
- **Appointments**: Total appointments set and completed
- **Quotes**: Total quotes created
- **Revenue**: Total sales revenue for the day

**Operations Metrics:**
- **Missed Stops**: Total missed stops (TMX + RNA)
- **Backlog**: Average backlog percentage

### Sales Team Performance
Month-to-date performance for each Account Executive:
- **Name and Email**: Contact information
- **Month Sales**: Total sales revenue for the month
- **Month TAP**: Total talk-to-appointments
- **Month Quotes**: Total quotes created
- **Ranking**: Sorted by sales performance (highest first)

### Operations Team Performance
Performance metrics for each technician:
- **Name and Email**: Contact information
- **Leads Submitted**: Total leads submitted this month
- **Installs Completed**: Total installations completed

### Branch Pipeline
Overview of your branch's sales pipeline:
- **Proposal Value**: Total value of opportunities in proposal stage
- **Negotiation Value**: Total value of opportunities in negotiation
- **Sold Value**: Total value of recently sold opportunities
- **Total Pipeline Value**: Combined value of all active opportunities

### Alerts
Automated alerts for items requiring attention:
- **Overdue Proposals**: Proposals older than 14 days
- **Unassigned Installations**: Sold deals without assigned specialists
- **High-Severity Issues**: Open high-priority service issues

Alerts are color-coded by severity:
- **High**: Red - Requires immediate attention
- **Medium**: Yellow - Should be addressed soon

### Week Trends
Week-over-week comparison:
- **This Week**: Total sales for current week
- **Last Week**: Total sales for previous week
- **Change**: Percentage change (positive or negative)

## Generating Reports

### Daily Cadence Report

Generate a daily summary report for your branch:

1. Navigate to "Generate Reports"
2. Select "Daily Cadence Report"
3. Click "Generate"
4. The system will:
   - Create a summary entry for today
   - Include all sales and operations metrics
   - Generate a report ID
   - Log the report generation

**Report Contents:**
- Today's sales summary
- Today's operations summary
- Team performance highlights
- Key metrics and trends

### Exporting Data

Export data for external analysis:

1. Navigate to "Export Data"
2. Select report type:
   - Sales Activity
   - Leads
   - Operations Metrics
   - Tracker Data
3. Apply filters (optional):
   - Date range
   - Team member
   - Status
4. Click "Export to CSV"
5. Download the CSV file

**Use Cases:**
- Excel analysis
- PowerPoint presentations
- Email reports
- External reporting systems

### Custom Reports

For custom reports or additional data:
- Contact your system administrator
- Use the API documentation
- Request specific report formats

## Managing Alerts

### Alert Types

**Overdue Proposals:**
- Proposals that haven't been followed up in 14+ days
- Shows customer name and days overdue
- Action: Follow up with AE, review opportunity status

**Unassigned Installations:**
- Sold deals without assigned specialists
- Shows customer name and sold date
- Action: Assign specialist, schedule installation

**High-Severity Issues:**
- Open service issues marked as high priority
- Shows customer name and issue type
- Action: Review issue, ensure resolution in progress

### Responding to Alerts

1. Review alerts on your dashboard
2. Click on an alert to view details
3. Take appropriate action:
   - **Overdue Proposals**: Contact AE, review opportunity
   - **Unassigned Installations**: Assign specialist immediately
   - **High-Severity Issues**: Verify resolution progress, escalate if needed
4. Alerts automatically update when underlying issues are resolved

### Alert Management Best Practices
- Review alerts daily
- Address high-severity items immediately
- Use alerts for coaching opportunities
- Track alert resolution times
- Identify patterns in alerts

## Coaching Workflows

### Using Data for Coaching

**Sales Coaching:**
1. Review individual AE performance
2. Identify areas for improvement:
   - Low TAP numbers
   - Low quote generation
   - Low win rates
   - Pipeline gaps
3. Schedule one-on-one coaching sessions
4. Use dashboard data to guide conversation
5. Set specific improvement goals
6. Follow up on progress

**Operations Coaching:**
1. Review technician performance
2. Identify areas for improvement:
   - Low lead submission
   - Installation delays
   - Issue frequency
3. Provide targeted feedback
4. Share best practices from top performers
5. Set improvement goals

### Coaching Session Structure

1. **Review Performance Data**
   - Show dashboard metrics
   - Compare to goals
   - Compare to team averages

2. **Identify Strengths**
   - Recognize what's working
   - Reinforce positive behaviors

3. **Identify Opportunities**
   - Discuss areas for improvement
   - Set specific, measurable goals

4. **Create Action Plan**
   - Define steps to improve
   - Set timeline
   - Schedule follow-up

5. **Document Session**
   - Record coaching notes
   - Track progress over time

### Performance Improvement Plans

For team members needing significant improvement:
1. Review performance trends
2. Identify root causes
3. Create structured improvement plan
4. Set clear milestones
5. Schedule regular check-ins
6. Provide additional support and resources
